
## Environment

Tested on **Ubuntu 20.04.6 LTS**.

### Step 1: Build Virtual Environment

```bash
python -m venv ./venv/
source ./venv/bin/activate
```

### Step 2: Install dependencies

```bash
pip install -r requirements.txt
```
